# Overview of buttons

Buttons are elements on which you can click and an action happens.
Buttons should be different from links. If you want to link somewhere use a link.
If you want to submit a form, use a button.
If you want to do some actions on page, use a button.

Styling of buttons coming from [Tailwind UI](https://tailwindui.com/components/application-ui/elements/buttons)
