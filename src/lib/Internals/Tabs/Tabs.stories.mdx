# Overview of tabs

Tabs are used mainly to categorize large amount of information and to add hiearchical structure to the content. They are also used to switch between different views of the same data.

Styling of tabs coming from [Tailwind UI](https://tailwindui.com/components/application-ui/navigation/tabs)
